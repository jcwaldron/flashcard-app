## Run Locally

Clone the project

```bash
  git clone https://github.com/jcwaldron/react-flashcards-app.git
```

Go to the project directory

```bash
  cd react-flashcards-app
```

Install dependencies

```bash
  npm install
```

Start the monorepo

```bash
  npm run start
```
